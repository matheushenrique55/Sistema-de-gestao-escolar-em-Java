Sistema de Gestão Escolar (Java)

Sistema de gestão escolar desenvolvido em Java, executado via terminal, com foco em orientação a objetos e manipulação de listas (ArrayList).

Permite o gerenciamento completo de alunos, professores, disciplinas e notas.

📌 Funcionalidades

✅ Cadastrar alunos

✅ Cadastrar professores

✅ Cadastrar disciplinas

✅ Vincular disciplinas aos alunos

✅ Lançar notas (3 avaliações por disciplina)

✅ Consultar notas individuais

✅ Listar alunos, professores e disciplinas

✅ Exibir panorama geral com média final e status (APROVADO/REPROVADO)

🧠 Conceitos aplicados

Programação Orientada a Objetos (POO)

Encapsulamento

Relacionamento entre classes

Uso de ArrayList

Estruturas de repetição e decisão

Manipulação de dados via Scanner

Organização modular do sistema

📂 Estrutura do Projeto

O projeto é composto pelas seguintes classes principais:

Main → Classe principal com menu interativo

Aluno → Representa os alunos cadastrados

Professor → Representa os professores

Disciplina → Representa as disciplinas

Boletim → Controla notas e cálculo da média

Pessoa → Classe base (herança)

Controle → Auxilia no controle de fluxo do sistema

📊 Regras de Avaliação

Cada disciplina possui 3 notas

Média final calculada automaticamente

Status:

🟢 APROVADO → média ≥ 6.0

🔴 REPROVADO → média < 6.0
